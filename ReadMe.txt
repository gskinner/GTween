The contents of this directory:

1) The "docs/" directory holds all documentation.

2) "docs.html" provides a shortcut to "index.html" in the "docs" directory.

3) This "ReadMe.txt" file.

4) The "src/" directory contains all of the source code for GTween.

5) The "samples/" directory contains the accompanying demos. These demos can be compiled in Flash Pro (using the .fla files) or Flex/Flash Builder (using the .as files).

6) The "swc/" directory contains the GTween SWC library with all of the GTween class files (the compiled contents of "src/").
